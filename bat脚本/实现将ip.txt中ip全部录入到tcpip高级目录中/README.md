## 脚本名称:自动化添加ip到windows网络高级目录
#### 功能描述:index.bat脚本可以将ip.txt中的ip和掩码按照顺序进行自动添加到windows系统的tcp/ip的高级目录中
##### 此脚本应一个学长所托编写，此脚本目前平稳运行在杭州地铁局办公电脑中

<hr>

#### 脚本执行:下载到本地windows系统中，直接点击即可，代码内置了管路员身份运行
